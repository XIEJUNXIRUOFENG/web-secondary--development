## 配置项
### id
**`字段描述`**
二开组件名称

**`样例`**
堆叠分组

### colorList
**`字段描述`**
系列颜色,不同颜色 使用|分隔开

**`样例`**
name|sex


### sechartStyle
**`字段描述`**
自定义筛选条件的列 的样式，与平台 的自定义样式类似

**`样例`**
{ .select_tile{background:red} }

### colorToptitle
**`字段描述`**
柱状顶部的文字颜色 

**`样例`**
red

### barMaxWidth
**`字段描述`**
柱状最大宽度 ,默认为60

**`样例`**
20
